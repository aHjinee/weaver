## 📌 프로젝트 소개

실을 엮어 하나의 천을 만드는 **WEAVER** 처럼
흩어진 부서와 직원 데이터를 통합하고 Spring Batch로 안전하게 보존하는 HR 데이터 관리 플랫폼입니다.

- 프로젝트 기간 : **2026.05.14 ~ 2026.05.26**
- 개발 인원 : **6명**
- 개발 방식 : **Github Flow 기반 협업**
